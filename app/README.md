# NeRF-Augmented ViT Training App

## Overview

This app provides an interactive platform for depth estimation and feature matching using state-of-the-art vision models. It is designed for research and experimentation, featuring both a web-based UI and a REST API.

## Used Technologies

- **Backend**:  
  - Python  
  - FastAPI (REST API)  
  - Gradio (interactive web UI)  
  - PyTorch (model inference)  
  - Depth-Anything-V2, Tiny RoMa (vision models)

- **Frontend**:  
  - React  
  - TypeScript  
  - Vite (build tool)  
  - Material UI (component library)

## Structure

- `backend/`: Python backend for model inference and API/UI.
- `frontend/`: React frontend for user interaction and visualization.

For setup, usage, and development details, see the respective `README.md` files in each subfolder.
